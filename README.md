# Option Toiture — Site Web

Site web statique pour Option Toiture, hébergé sur Cloudflare Pages.

## Structure
```
/
├── index.html                  → Page d'accueil
├── installation-rona.html      → Page RONA+ (à venir)
├── decouvrir-option-toiture.html → Page Découvrir (à venir)
```

Le site se déploie automatiquement sur Cloudflare Pages à chaque push sur `main`.
